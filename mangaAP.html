Projet manga

Contexte :

Au cours de notre deuxième année on a eu l'occasion de mettre en application nos connaissances qu'on a acquis lors du début d'année pour réaliser une application avec larvel.
J'ai donc utilisé le modele MVC.

Le cahier des charges :

- Réusir à afficher les images
- Lister dans un select pour l'ajout.
- Ajouter un manga et pouvoir le modifier

Technologie utilisé :

  • PHP
  • Laravel
  • SQL/Eloquent (pour l'interaction avec la BD)
  • HTML/CSS/Bootstrap

Bilan :

J'ai rencontré quelque difficulté lors de ce projet notamment sur l'affichage de l'image dans la liste des mangas.
Je n'arrivais pas à afficher les images et afficher l'image erreur quand il n'avait pas d'image dans la bd. Pour résoudre le problème je me suis renseigné sur le internet pour trouver la solution.
